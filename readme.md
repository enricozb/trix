# Trix

A nix flake to generate a rust module with tree-sitter languages.

## Why

1. Lets you reference the `grammar.js` files directly, so you can make use of
   open source grammars where rust bindings where not generated by the author.
2. Generates the tree-sitter bindings at build-time, so there is no chance of
   tree-sitter version mismatch.
3. Doesn't pollute your `Cargo.toml` dependencies (just your nix flake inputs).
